What is applet in java?

Applets are small Java applications which can be accessed on an Internet server, transported over the Internet, and can be installed and run automatically as part of a web document. The applet can create a graphical user interface after a user gets an applet. It has restricted access to resources so that complicated computations can be carried out without adding the danger of viruses or infringing data integrity. Any Java applet is a class that extends the class of java.applet.Applet. There is no main() methods in an Applet class. Using JVM it is regarded. The JVM can operate an applet application using either a Web browser plug-in or a distinct runtime environment. JVM generates an applet class instant and invokes init() to initialize an applet.
